# 7 Use Cases mới của Claude — Tóm tắt

> **Tác giả:** YouTuber AI (không rõ tên)

## Tổng quan

Video giới thiệu 7 tính năng mới của Claude (Anthropic) và cách ứng dụng thực tế cho doanh nghiệp online. Claude không còn chỉ là chatbot — mà là hệ thống tự động hóa công việc end-to-end.

## 7 Tính năng & Use Cases

### 1. Memory 2.0 (Autodream)

- Claude tự tổng hợp bộ nhớ từ các cuộc hội thoại mỗi 24 giờ
- Autodream: giữa các phiên, Claude dọn dẹp, gộp, cập nhật bộ nhớ
- Nền tảng cho mọi tác vụ tự động — Claude hiểu business của bạn theo thời gian

### 2. Co-work Projects — Weekly CEO Review

- Giao task cho Claude, Claude tự thực hiện (khác với chat thông thường)
- **Có bộ nhớ giữa các lần chạy** → mỗi lần chạy thông minh hơn
- Use case: Weekly CEO Review — Claude tự kéo metrics, so sánh tuần trước, báo cáo delta
- 3 cách tạo: từ Co-work, import từ Projects, hoặc từ folder có sẵn

### 3. Dispatch — Điều khiển từ xa

- Giao task cho Claude từ điện thoại, Claude làm việc trên máy tính
- Máy tính phải bật + Claude desktop mở (có chế độ Keep Awake)
- Use case: "Nghiên cứu 3 đối thủ, tạo bảng so sánh giá" → giao tối, sáng nhận kết quả
- Use case: Dọn dẹp thư mục tự động

### 4. Dispatch + Computer Use — YouTube Analytics

- Kết hợp Dispatch với Computer Use → Claude thao tác trực tiếp trên trình duyệt
- Use case: Từ điện thoại, yêu cầu Claude vào YouTube Analytics, kéo data 10 video gần nhất, tạo bảng phân tích
- Cần đăng nhập sẵn các ứng dụng trên máy tính

### 5. Scheduled Tasks — AI News Digest

- Lên lịch task chạy định kỳ (hàng ngày, hàng tuần...)
- Use case: Claude đọc 18 newsletter AI trong Gmail, tổng hợp, gửi digest vào Slack mỗi sáng 7h
- Sử dụng Connectors (Gmail, Slack) để kết nối các ứng dụng

### 6. Computer Use — UX Audit tự động

- Claude mở app trong trình duyệt, đi qua toàn bộ user flow
- Tự đăng nhập (kể cả lấy OTP từ email), test từng bước
- Tạo báo cáo UX audit đầy đủ trong <15 phút (thay vì nửa ngày)

### 7. Visualizations — Flowchart tương tác

- Claude tạo biểu đồ/flowchart tương tác ngay trong chat
- Có thể click từng bước, xem chi tiết
- Export qua Mermaid Chart connector → nhúng vào Notion, website
