# Nhật ký hàng ngày — Tony Hoang

## Cấu trúc thư mục

```
docs/nhat-ky/
├── README.md          ← File này
├── 2026-04/           ← Thư mục theo tháng
│   ├── 2026-04-08.md
│   ├── 2026-04-09.md
│   └── ...
├── 2026-05/
└── ...
```

## Hướng dẫn

- Mỗi ngày 1 file, đặt tên theo format: `YYYY-MM-DD.md`
- Điền vào cuối ngày (hoặc sáng hôm sau khi review)
- Không cần điền hết — chỉ điền mục có hoạt động
- Claude sẽ dùng dữ liệu này để phân tích xu hướng và đề xuất điều chỉnh
